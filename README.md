# 致刘子梁的告白网站

这是一个专属于刘子梁的告白网站，包含个性化的欢迎页面、表达心意的文字内容、照片展示区和互动元素。

## 功能特点

- **个性化欢迎页面**：带有动画效果的欢迎界面
- **心意文字内容**：表达真挚情感的文字
- **照片展示区**：展示美好回忆的照片墙
- **互动元素**：点击查看祝福卡片和浮动爱心动画
- **响应式设计**：适配各种设备屏幕尺寸

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 动画)
- JavaScript (ES6+)
- Google Fonts

## 如何部署

### 本地运行

1. 克隆或下载本仓库
2. 用浏览器打开 `index.html` 文件即可查看

### GitHub Pages 部署

1. 在GitHub上创建一个新仓库
2. 将代码推送到仓库
3. 进入仓库设置，找到 GitHub Pages 选项
4. 选择 `main` 分支作为源，点击保存
5. 等待几分钟，网站将部署在 `https://yourusername.github.io/repositoryname`

## 自定义内容

1. **修改文字内容**：编辑 `index.html` 文件中的 `<section id="message">` 部分
2. **添加照片**：将照片替换 `photos-section` 中的占位符
3. **更改样式**：修改 `style.css` 文件中的颜色、字体和布局
4. **调整动画**：修改 `script.js` 文件中的动画效果
